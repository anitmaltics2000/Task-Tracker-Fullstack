# Task Tracker Frontend

This is the frontend for the Task Tracker application, built with React. It provides the user interface to manage tasks.

## Prerequisites

- Node.js
- npm

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at **http://localhost:3000**.

> **Note:** Ensure the backend server is already running on port 8000 before running the frontend for the application to function correctly.

## Features

- **View Tasks:** See a list of all tasks.
- **Add Task:** Create a new task with a date and reminder setting.
- **Delete Task:** Remove a task from the list.
- **Toggle Reminder:** Double-click a task to toggle the reminder highlight.
